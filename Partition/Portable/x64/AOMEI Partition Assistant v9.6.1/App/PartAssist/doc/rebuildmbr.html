Rebuild MBR

Rebuild MBR could fix the corrupt Master Boot Record, let computer boot correctly. It may help you solve the problem like "operating system not found" or blank screen.

To rebuild MBR

  1. Select the disk and right-click the disk select Rebuild MBR;
  2. Set the type of MBR for current OS;
  3. Click Apply on the toolbar to commit the changes.

Note:
This feature is usually used on system disk. When you encounter system could not boot, you'll need to connect system disk to another machine, then in the toolbar select Wizard -> Make Bootable CD Wizard to build a bootable device to fix this problem.
