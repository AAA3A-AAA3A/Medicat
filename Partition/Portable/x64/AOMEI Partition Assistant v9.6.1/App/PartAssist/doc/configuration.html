Working Computer Configuration
Minimum Hardware requirements:
  • 500 MHz x86 or compatible CPU
  • 256MB RAM memory
  • Mouse and Keyboard
Minimum Hardware requirements:
  • 1 GHz x86 or compatible CPU
  • 512mb RAM memory
  • Mouse and Keyboard
Partition Assistant supports all the storage devices which can be recognized by Windows, such as IDE, SATA, SCSI, SSD, USB external disks, flash drive, Fire wire disk, Hardware RAID and Flash drive and so on. It is able to support MBR (under 2TB) and GPT (larger than 2TB) with 128 hard disks, with file system FAT, FAT32, NTFS and also EXT2, EXT3, EXT4.

It supports different versions of Windows depending on different software editions. In general, it works on Windows 2000/XP/Vista/7/8/8.1/10, Small Business Server 2011/2003/2008, Windows home server 2011 and Windows Server 2003/2008/2008(R2)/2012/2012(R2)/2016 (both 32bit and 64bit).
