Wipe Hard Drive

Wipe hard drive will delete all partitions and erase the data completely to ensure the data can not be recovered by any data recovery solution. This function can prevent undesired recovery of your deleted data.

To wipe hard drive

  1. In the disk map, select the disk which you want to wipe;
  2. By right clicking the disk select Wipe Hard Drive;
  3. In the popup window, enter a proper value. This value means how many times you want to wipe this hard drive. Click OK to continue;
  4. Click Apply button on the toolbar to commit this operation.
Note:
You can specify the number of times to wipe from 1 to 100.
