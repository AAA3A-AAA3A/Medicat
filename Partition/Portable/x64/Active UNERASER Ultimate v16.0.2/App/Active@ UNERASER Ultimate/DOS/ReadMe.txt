This is related to a a bootable CD/DVD Image containing FreeDOS and 
Active@ UNERASER for DOS.

Windows Console application UNERASE.EXE is accessible under Windows.


UNERASER.EXE is located on bootable partition of this CD-ROM and thus is 
not accessible under Windows.


Bootable CD-ROM operations:

To make it work under DOS:
- Insert this CD-ROM into a CD-ROM drive of the computer to be erased
- Check that the CD-ROM has a boot priority over HDD in BIOS settings
- Re-boot the machine
- Computer should boot in DOS mode and Active@ UNERASER is launched at startup.


To make it work under Windows:
- Insert this CD-ROM into a CD-ROM drive of the computer
- Locate the file UNERASE.EXE on the CD-ROM drive
- Run UNERASE.EXE to start the Windows Console version of UNERASER
